1. Open a Terminal and navigate to the server directory within chitchat.
2. Run the command:go run server.go
3. When the server is running, open a terminal for each user you want have in the chat.
4. Each user must navigate to the client directory through chitchat and then into client.
5. Run the command:go run client.go
6. Every client/user will be required to give a username before entering. Type in whatever name you wish.
7. You are not free to chat!
8. When you wish to leave the chat press: Ctrl + C
